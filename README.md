# FastSnackLasProgramadoras
Actividad de clase para aplicar tipos de clases (2 tipos de clase, abstractas, completas)

El objetivo principal del proyecto es desarrollar una herramienta que permita administrar clientes, productos, pedidos, facturas y pagos de forma organizada. 
Además, buscamos agilizar la atención al cliente mediante el almacenamiento de información en una base de datos, permitiendo mantener registros actualizados de las compras realizadas y facilitando la gestión administrativa y contable del negocio.
